# 个人交易习惯
一般持有2个月-6个月，交易上一个月可能交易10次左右，超过10个点浮亏会考虑止损
看好的点位会分批买入，高点也会分批卖出
下面设计一套适合我的策略打分器

# 回测数据如何选择以及如何切分？

## 数据选择
选取bundle数据集里的最近10年的数据来跑回测，这10年的数据进行分段，每段1年，步长3个月，往前滚动

## 基准指数
统一使用沪深300（000300.XSHG）作为基准，用于：
- 策略超额收益、超额回撤、跟踪误差、信息比率等指标的计算
- 月度超额胜率的计算
- 市场环境标签（牛市/熊市/震荡）的判定

打分器运行回测时强制带上 `--benchmark 000300.XSHG`，确保所有依赖基准的指标均可计算。

## 切分设计
数据范围：最近 10 年（约 2016-02 ~ 2026-02）
窗口大小：1 年
步长：3 个月
窗口数量：37 个，(120 - 12) / 3 + 1 = 37 个窗口

边缘季度处理：数据范围两端的季度可能不完整（如Q1 2016只有2~3月），只要该季度有至少1个窗口覆盖即纳入计算。季度得分来自覆盖它的窗口分数的平均值，窗口本身是完整1年回测，不存在数据缺失。

# 评判策略的指标（选取最核心的指标如下）
## 收益类指标：
- total_returns — 这个窗口里实际赚了多少
- annualized_returns — 考虑了时间效率，赚同样的钱用时越短，年化越高，代表赚的快不快
- excess_annual_returns — 比基准多赚多少，有没有跑赢市场

## 风险类指标：
- max_drawdown 最大回撤，亏多深
- max_drawdown_duration_days 最大回撤持续时间，回撤持续多久
- excess_max_drawdown 超额最大回撤，你相对基准最多落后多少。指数增强的命脉就是超额不能大幅回撤
- tracking_error 跟踪误差，你的策略和基准偏离多大。指数增强要求不能偏离太远

## 风险调整收益类：
- sharpe — 夏普率，每承受一份风险赚多少收益，最通用的策略评价指标，越高越好，代表赚的多，但颠簸少
- sortino — 索提诺比率，只惩罚下跌波动不惩罚上涨波动
- information_ratio — 信息比率，每承受一份跟踪误差赚多少超额收益，做指数增强的核心指标

## 交易统计指标：
- win_rate — 日胜率，多少天是赚钱的
- profit_loss_rate — 盈亏比，赚的时候平均赚多少 / 亏的时候平均亏多少

## 月度指标：
- monthly_excess_win_rate — 每个月的超额日胜率，每个月的超额收益是否赚钱

# 打分方法

## 评分函数
使用4次多项式，5个锚点精确拟合，中间平缓、两端加速：
```
锚点范围内：score = c4*x⁴ + c3*x³ + c2*x² + c1*x + c0
锚点范围外：score = y_端点 + slope * (x - x_端点)
```
- 对于"越低越好"的指标，先取反 x' = -x，再代入多项式
- 锚点范围外使用线性外推，保证单调性（好的更好，差的更差）
- 外推斜率取多项式在端点的导数值，但不低于平均斜率的50%（最小斜率保障）

# 窗口聚合方法

## 方案：季度网格投影 + 时间衰减

解决重叠窗口（步长3个月，窗口1年，重叠75%）导致同一段行情被多个窗口重复计分的问题。

### 步骤

**第一步：计算窗口分数**
正常计算37个窗口的综合分（13个指标加权求和），得到 score_1 ~ score_37。

**第二步：建立季度网格**
10年 = 40个季度（Q1 2016 ~ Q4 2025）。每个季度被若干窗口覆盖。

**第三步：投影到季度**
每个季度的得分 = 覆盖该季度的所有窗口分数的平均值。
```
quarterly_score[Q] = mean(所有包含季度Q的窗口分数)
```
这样每段行情不管被几个窗口评估，最终只贡献一份权重。

**第四步：时间衰减加权**
对40个季度得分做指数衰减加权：
```
权重_i = exp(-λ × i)
```
其中 i=0 是最近的季度，i=39 是最远的季度，λ=0.03。
最近季度 vs 最远季度权重比约 3:1，近2年贡献约50%权重。

**第五步：计算最终分数**
```
最终分数 = Σ(quarterly_score[i] × 权重_i) / Σ(权重_i)
```

# 稳定性评估

独立于综合得分，不混入总分。
使用窗口聚合第三步的 quarterly_score（投影后、时间衰减前的原始季度得分）进行计算。

## 稳定性评分（0~100分）

三个维度加权合成：

| 维度 | 权重 | 0分 | 50分 | 100分 |
|---|---|---|---|---|
| CV（变异系数） | 50% | CV ≥ 0.5 | CV = 0.25 | CV ≤ 0.1 |
| 最差季度得分 | 30% | ≤ -10分 | 25分 | ≥ 50分 |
| 最长连续低分季度数 | 20% | ≥ 5个 | 2个 | 0个 |

各维度线性插值，超出范围按边界值（0或100）。
```
稳定性分数 = 0.5 × CV分 + 0.3 × 最差季度分 + 0.2 × 连续低分分
```

其中"低分"阈值为30分。

备注：当季度平均分接近0时，CV会极大，但CV ≥ 0.5直接给0分，结果合理，无需额外处理。

# 市场环境适应性

## 市场环境标签

用基准指数的季度涨跌幅划分：

| 基准季度涨跌幅 | 标签 |
|---|---|
| > 8% | 牛市 |
| < -8% | 熊市 |
| -8% ~ 8% | 震荡 |

## 计算方式

按市场环境分组，分别计算该环境下所有季度得分的简单平均值（不做时间衰减）。
使用窗口聚合第三步的 quarterly_score（投影后、时间衰减前的原始季度得分）。

# 最终输出格式

```
【策略综合得分】 xx.x 分 | 收益 + 风险 + 指标加权总分
【策略稳定得分】  xx 分 | 策略综合得分相同时，反应策略的泛化性
【市场环境】牛市 xx.x | 震荡 xx.x | 熊市 xx.x
```
